# Computação Quântica / Criptografia Quântica

## [Wiki](https://github.com/alberthydev/quantum-cryptography/wiki)


### Sobre

Esse repositório será usado para armazenar referências bibliográficas da minha pesquisa sobre criptografia quântica, tudo e qualquer conhecimento desenvolvido para essa pesquisa será armazenada neste repositório a efeito de documentação do progresso, visando achar falhas em minha forma de pesquisa, a ideia é otimizar o processo para futuros títulos como pos-graduação, mestrado ou doutorado.
